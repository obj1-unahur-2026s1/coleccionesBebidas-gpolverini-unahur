# Changelog

Todos los cambios notables en este proyecto serán documentados en este archivo.

El formato está basado en [Keep a Changelog](https://keepachangelog.com/es-ES/1.0.0/),
y este proyecto adhiere a [Semantic Versioning](https://semver.org/lang/es/).

## [1.0.0] - 2026-04-29

### Agregado
- Sistema de voluntario (Tito) con gestión de consumo de bebidas y cálculo de velocidad
- Tres tipos de bebidas con diferentes efectos: whisky, tereré y cianuro
- Suite completa de tests con formato BDD y cobertura del 100%
- Documentación técnica completa (arquitectura, desarrollo, setup, testing)
- Guías de contribución y código de conducta
- Configuración del proyecto para Wollok con estructura estándar
